15 Modules

Everything you need,
nothing you don't

Store

Document Storage

Soft-delete, 8 persistent indexes, entity and category filtering. All writes are immutable dataclasses.

Vector

Semantic Search

BAAI/bge-m3 embeddings with numpy cosine similarity. Auto-upgrades to ArangoDB native vector index when available.

CDC

Change Data Capture

Every insert, update, and delete tracked in a changelog with revision-based consumer checkpoints and TTL cleanup.

Audit

Audit Logging

Full operation history with agent, session, and workflow context. Auto-expires with configurable TTL retention.

Scope

Agent Scoping

Private, workflow, and global visibility levels. Control exactly which agents and workflows can see which records.

Temporal

Supersession Chains

Track how facts evolve. Walk supersession chains, find current versions, detect contradictions across entities.

Graph

Graph Traversals

SmartGraph-like partition routing with concurrent.futures parallel traversals from multiple start vertices.

Events

Event Bus

Pluggable pub/sub with in-process, Redis, and NATS backends. React to memory.created, updated, deleted in real time.

Retrieval

Multi-layer Retrieval

4-layer cascade: exact tag match, semantic vector, temporal range, and graph expansion. Fused with reciprocal rank.

Quick Start

Five lines to production-ready

main.py 
from open_argondb import ArgonDB
from open_argondb.models import Memory, AgentScope

# Connect — audit and CDC enabled by default
db = ArgonDB(host="http://localhost:8529", database="myapp")

# Insert with agent scoping
scope = AgentScope(agent_id="agent-1", workflow_id="wf-123")
mem = Memory(id="m1", content="User prefers dark mode", tags=["preference"])
db.insert(mem, scope=scope)

# Embed and search
db.batch_embed()
results = db.search("what does the user prefer?", limit=10)

# CDC — get changes since last checkpoint
changes = db.get_changes()

# Supersede outdated facts
new = Memory(id="m2", content="User switched to light mode", tags=["preference"])
db.supersede("m1", new, scope=scope)
